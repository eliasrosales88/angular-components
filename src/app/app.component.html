Databinding (avanzado)

Decorador @Input | Hijo -> Padre

El decorador @Input sirve para conectar/unir las propiedades de un componente con otro. Esto quiere decir que la informacion almacenada en una propiedad de un componente, viaja a otro para que aquel que la recibe pueda usar la informacion.

Este decorador se usa usualmanete para pasar informacion de un componente hijo a padre

Para saber cual es un componente hijo y cual es un componente padre solo tenemos que fijarnos en cual de los dos componentes, uno contiene las etiquetas del otro dentro de su plantilla html.
