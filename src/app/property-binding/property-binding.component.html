El siguiente boton tiene el atributo html "disabled" pero con el property-binding de Angular de esta manera el codigo html queda asi:

< button class="btn btn-primary" [disabled]="isDisabled" >Property binding< /button >

El codigo en el archivo TS del componente queda de la siguiente manera:

{{'export class PropertyBindingComponent implements OnInit {'}}
isDisabled = {{isDisabled}};

El metodo que hace que cambie a los 2 segundos es el siguiente:

{{'constructor() {'}}
{{'setTimeout(() =>{'}}
{{'this.isDisabled = false;'}}
{{'}, 2000);'}}
